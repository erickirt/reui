## API Reference

This section documents the public API surface of the Filters component.

### Filters

The main Filters component provides a comprehensive filtering system with support for multiple field types, operators, and visual indicators.

| **Prop**                                                                                           | **Type**                                                                                                                        | **Default** |
| -------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------- | ----------- |
| `filters` <PopoverInfo>Array of active filters to display.</PopoverInfo>                           | ~Array~ <PopoverInfo> Array\<Filter\<T\>> </PopoverInfo>                                                                        | <Na/>       |
| `fields` <PopoverInfo>Field configuration for available filter options.</PopoverInfo>              | ~FilterFieldsConfig\<T\>~ <PopoverInfo> Union type supporting both flat arrays and grouped field configurations. </PopoverInfo> | <Na/>       |
| `onChange` <PopoverInfo>Callback function triggered when filters change.</PopoverInfo>             | ~(filters: Filter\<T\>[]) => void~ <PopoverInfo> Function that receives the updated filters array. </PopoverInfo>               | <Na/>       |
| `className` <PopoverInfo>Additional class names for styling the filters container.</PopoverInfo>   | ~string~                                                                                                                        | <Na/>       |
| `showAddButton` <PopoverInfo>Whether to show the add filter button.</PopoverInfo>                  | ~boolean~                                                                                                                       | ~true~      |
| `addButtonText` <PopoverInfo>Custom text for the add filter button.</PopoverInfo>                  | ~string~                                                                                                                        | <Na/>       |
| `addButtonIcon` <PopoverInfo>Custom icon for the add filter button.</PopoverInfo>                  | ~ReactNode~                                                                                                                     | <Na/>       |

### Field Types

The Filters component supports various field types, each with specific behaviors and operators:

| **Field Type** | **Operators**                                                                                                                        | **Input Type**     | **Description**                                            |
| -------------- | ------------------------------------------------------------------------------------------------------------------------------------ | ------------------ | ---------------------------------------------------------- |
| `text`         | ~enum~ <PopoverInfo> "contains" \| "not_contains" \| "starts_with" \| "ends_with" \| "is" \| "empty" \| "not_empty" </PopoverInfo>   | ~Text Input~       | Text input with optional pattern validation and search.    |
| `number`       | ~enum~ <PopoverInfo> "equals" \| "not_equals" \| "greater_than" \| "less_than" \| "between" \| "empty" \| "not_empty" </PopoverInfo> | ~Number Input~     | Number input with min/max/step and optional prefix/suffix. |
| `date`         | ~enum~ <PopoverInfo> "before" \| "after" \| "is" \| "is_not" \| "empty" \| "not_empty" </PopoverInfo>                                | ~Date Picker~      | Calendar-based date selection.                             |
| `daterange`    | ~enum~ <PopoverInfo> "between" \| "not_between" \| "empty" \| "not_empty" </PopoverInfo>                                             | ~Date Range~       | Two date pickers with "to" separator.                      |
| `select`       | ~enum~ <PopoverInfo> "is" \| "is_not" \| "empty" \| "not_empty" </PopoverInfo>                                                       | ~Dropdown~         | Searchable dropdown with predefined options.               |
| `multiselect`  | ~enum~ <PopoverInfo> "is_any_of" \| "is_not_any_of" \| "includes_all" \| "excludes_all" \| "empty" \| "not_empty" </PopoverInfo>     | ~Multi-Select~     | Multi-select dropdown with search and limits.              |
| `boolean`      | ~enum~ <PopoverInfo> "is" \| "is_not" \| "empty" \| "not_empty" </PopoverInfo>                                                       | ~Switch~           | Toggle switch with custom on/off labels.                   |
| `email`        | ~enum~ <PopoverInfo> "contains" \| "not_contains" \| "starts_with" \| "ends_with" \| "is" \| "empty" \| "not_empty" </PopoverInfo>   | ~Email Input~      | Email input with built-in validation.                      |
| `url`          | ~enum~ <PopoverInfo> "contains" \| "not_contains" \| "starts_with" \| "ends_with" \| "is" \| "empty" \| "not_empty" </PopoverInfo>   | ~URL Input~        | URL input with validation.                                 |
| `tel`          | ~enum~ <PopoverInfo> "contains" \| "not_contains" \| "starts_with" \| "ends_with" \| "is" \| "empty" \| "not_empty" </PopoverInfo>   | ~Phone Input~      | Phone input with validation.                               |
| `time`         | ~enum~ <PopoverInfo> "before" \| "after" \| "is" \| "between" \| "empty" \| "not_empty" </PopoverInfo>                               | ~Time Picker~      | Time selection component.                                  |
| `datetime`     | ~enum~ <PopoverInfo> "before" \| "after" \| "is" \| "between" \| "empty" \| "not_empty" </PopoverInfo>                               | ~DateTime Picker~  | Combined date and time selection.                          |
| `custom`       | ~Custom~                                                                                                                             | ~Custom Component~ | Provided by `customRenderer`.                              |
| `separator`    | ~None~                                                                                                                               | ~Visual Separator~ | Non-interactive separator.                                 |
