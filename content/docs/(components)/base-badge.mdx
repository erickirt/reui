## API Reference

This component supports both Base UI's `useRender` hook pattern and traditional `asChild` prop for backward compatibility.

### Badge

| **Prop**                                                                                                                                  | **Type**                                                                                                                       | **Default** |
| ----------------------------------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------ | ----------- |
| `render` <PopoverInfo>Element or component to render the badge as.</PopoverInfo>                                                          | ~React.ReactElement~                                                                                                           | `<span />`  |
| `asChild` <PopoverInfo>Change the default rendered element for the one passed as a child, merging their props and behavior.</PopoverInfo> | ~boolean~                                                                                                                      | `false`     |
| `variant` <PopoverInfo>The visual style variant of the badge.</PopoverInfo>                                                               | ~enum~ <PopoverInfo> "primary" \| "secondary" \| "success" \| "warning" \| "info" \| "outline" \| "destructive" </PopoverInfo> | `primary`   |
| `appearance` <PopoverInfo>The appearance style of the badge.</PopoverInfo>                                                                | ~enum~ <PopoverInfo> "default" \| "light" \| "outline" \| "ghost" </PopoverInfo>                                               | `default`   |
| `size` <PopoverInfo>The size of the badge.</PopoverInfo>                                                                                  | ~enum~ <PopoverInfo> "xs" \| "sm" \| "md" \| "lg" </PopoverInfo>                                                               | `md`        |
| `shape` <PopoverInfo>The shape of the badge.</PopoverInfo>                                                                                | ~enum~ <PopoverInfo> "default" \| "circle" </PopoverInfo>                                                                      | `default`   |
| `disabled` <PopoverInfo>Whether the badge is disabled.</PopoverInfo>                                                                      | ~boolean~                                                                                                                      | `false`     |

### BadgeButton

| **Prop**                                                                                                                                  | **Type**             | **Default**  |
| ----------------------------------------------------------------------------------------------------------------------------------------- | -------------------- | ------------ |
| `render` <PopoverInfo>Element or component to render the badge button as.</PopoverInfo>                                                   | ~React.ReactElement~ | `<button />` |
| `asChild` <PopoverInfo>Change the default rendered element for the one passed as a child, merging their props and behavior.</PopoverInfo> | ~boolean~            | `false`      |

### BadgeDot

This is a simple span element with no additional props beyond standard HTML attributes.
