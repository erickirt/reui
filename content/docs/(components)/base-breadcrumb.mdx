## API Reference

This component provides a customizable breadcrumb system for displaying a hierarchy of links.

### Breadcrumb

This component is a flexible container for displaying breadcrumb items.

| **Prop**                                                                                            | **Type**          | **Default**       |
| --------------------------------------------------------------------------------------------------- | ----------------- | ----------------- |
| `className` <PopoverInfo>Additional class names for styling the breadcrumb container.</PopoverInfo> | ~string~          | <Na/>             |
| `separator` <PopoverInfo>The character or element used as a separator.</PopoverInfo>                | ~React.ReactNode~ | `<ChevronRight/>` |

### BreadcrumbList

This component is used to display a list of breadcrumb items.

| **Prop**                                                                                       | **Type** | **Default** |
| ---------------------------------------------------------------------------------------------- | -------- | ----------- |
| `className` <PopoverInfo>Additional class names for styling the breadcrumb list.</PopoverInfo> | ~string~ | <Na/>       |

### BreadcrumbItem

This component is used to display an individual breadcrumb item.

| **Prop**                                                                                       | **Type** | **Default** |
| ---------------------------------------------------------------------------------------------- | -------- | ----------- |
| `className` <PopoverInfo>Additional class names for styling the breadcrumb item.</PopoverInfo> | ~string~ | <Na/>       |

### BreadcrumbLink

This component is used to display a link within a breadcrumb item.

| **Prop**                                                                                              | **Type**  | **Default** |
| ----------------------------------------------------------------------------------------------------- | --------- | ----------- |
| `className` <PopoverInfo>Additional class names for styling the breadcrumb link.</PopoverInfo>        | ~string~  | <Na/>       |
| `href` <PopoverInfo>The URL that the breadcrumb link points to.</PopoverInfo>                         | ~string~  | ~#~         |
| `asChild` <PopoverInfo>Renders the link as a child element, using the `Slot` component.</PopoverInfo> | ~boolean~ | `false`     |

### BreadcrumbPage

This component is used to display the current page in the breadcrumb.

| **Prop**                                                                                       | **Type** | **Default** |
| ---------------------------------------------------------------------------------------------- | -------- | ----------- |
| `className` <PopoverInfo>Additional class names for styling the breadcrumb page.</PopoverInfo> | ~string~ | <Na/>       |

### BreadcrumbSeparator

This component is used to display a separator between breadcrumb items.

| **Prop**                                                                                            | **Type** | **Default** |
| --------------------------------------------------------------------------------------------------- | -------- | ----------- |
| `className` <PopoverInfo>Additional class names for styling the breadcrumb separator.</PopoverInfo> | ~string~ | <Na/>       |

### BreadcrumbEllipsis

This component is used to display an ellipsis in the breadcrumb.

| **Prop**                                                                                           | **Type** | **Default** |
| -------------------------------------------------------------------------------------------------- | -------- | ----------- |
| `className` <PopoverInfo>Additional class names for styling the breadcrumb ellipsis.</PopoverInfo> | ~string~ | <Na/>       |
