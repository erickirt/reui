## API Reference

This component provides a versatile and customizable button system with various styles, appearances, and sizes.

### Button

This is the root component for creating buttons with multiple variants, appearances, sizes, and shapes.

| **Prop**                                                                                               | **Type**                                                                                                                                                          | **Default** |
| ------------------------------------------------------------------------------------------------------ | ----------------------------------------------------------------------------------------------------------------------------------------------------------------- | ----------- |
| `variant` <PopoverInfo>Specifies the button's theme.</PopoverInfo>                                     | ~enum~ <PopoverInfo> "primary" \| "mono" \| "destructive" \| "secondary" \| "outline" \| "dashed" \| "ghost" \| "dim" \| "foreground" \| "inverse" </PopoverInfo> | ~primary~   |
| `appearance` <PopoverInfo>Defines the visual style of the button.</PopoverInfo>                        | ~enum~ <PopoverInfo> "ghost" </PopoverInfo>                                                                                                                       | <Na/>       |
| `size` <PopoverInfo>Determines the size of the button.</PopoverInfo>                                   | ~enum~ <PopoverInfo> "md" \| "sm" \| "xs" \| "xxs" </PopoverInfo>                                                                                                 | ~md~        |
| `shape` <PopoverInfo>Specifies the shape of the button.</PopoverInfo>                                  | ~enum~ <PopoverInfo> "default" \| "circle" </PopoverInfo>                                                                                                         | ~default~   |
| `mode` <PopoverInfo>Defines the button's interaction style.</PopoverInfo>                              | ~enum~ <PopoverInfo> "default" \| "icon" \| "link" </PopoverInfo>                                                                                                 | ~default~   |
| `underline` <PopoverInfo>Sets the underline style for link mode.</PopoverInfo>                         | ~enum~ <PopoverInfo> "solid" \| "dashed" </PopoverInfo>                                                                                                           | <Na/>       |
| `underlined` <PopoverInfo>Defines a persistent underline for link mode.</PopoverInfo>                  | ~enum~ <PopoverInfo> "solid" \| "dashed" </PopoverInfo>                                                                                                           | <Na/>       |
| `asChild` <PopoverInfo>Renders the button as a child element using the `Slot` component.</PopoverInfo> | ~boolean~                                                                                                                                                         | ~false~     |
| `className` <PopoverInfo>Additional class names for styling the button.</PopoverInfo>                  | ~string~                                                                                                                                                          | <Na/>       |
